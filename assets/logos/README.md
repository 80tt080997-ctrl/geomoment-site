- 채팅 데모 헤더 → ChatGPT 로고 PNG로 교체

## 권장 사양

- 형식: PNG (투명 배경)
- 크기: 256×256 또는 512×512 (정사각)
- 비율: 1:1
- 코너: 라운드(rounded square) 또는 정사각 (CSS에서 처리)
